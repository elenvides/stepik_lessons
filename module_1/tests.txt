Тестовый сценарий 1.1.1 Поиск книги по названию
Предусловия:
1. Иметь книгу в каталоге, сохранить ее название (пример: The shellcoder's handbook )
2. Открыть главную страницу
Шаги:
- 1. В строке поиска ввести название книги
- 2. Нажать кнопку "Найти"
-- Открылась страница результатов поиска (в урле есть главная + search)
-- Книга с точным совпадением названия есть в результатах поиска
   (комментарий: сверять названия всех результатов на совпадение нельзя,
   так как ищет в том числе по описанию товара)


Тестовый сценарий 1.2.1 Регистрация пользователя с корректными данными
1. Иметь\сгенерировать почту, которая ранее не использовалась для регистрации
2. Иметь или сгенерировать пароль (Большие + маленькие буквы + цифры. Пример: 3gM3mpYmimqpx7b )
2. Открыть главную страницу
Шаги:
- 1. Нажать кнопку "Войти или зарегистрироваться"
-- Открылась страница регистрации-авторизации ( главная + accounts/login/ )
- 2. Заполнить поле "Адрес электронной почты": ввести подготовленный email
- 3. Заполнить поле "Пароль": ввести подготовленный пароль
- 4. Заполнить поле "Повторите пароль": ввести подготовленный пароль повторно
- 5. Нажать кнопку "Зарегистрироваться"
-- Переадрессация на главную страницу
-- Сообщение с текстом "Спасибо за регистрацию!"
   (комментарий: можно пробовать проверять без привязки к тексту,
                 на наличие не пустого єлемента с классом alert-success )
-- Пользователь авторизован (есть элементы):
   - иконка пользователя icon-user
   - ссылка на аккаунт href="/ru/accounts/"
   - кнопка выйти )
